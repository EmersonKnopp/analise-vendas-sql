# 📊 Análise de Performance de Vendas (SQL)

## 📌 Sobre o Projeto
Este projeto foca na extração e análise de indicadores financeiros de um e-commerce. O objetivo principal foi calcular o faturamento mensal e a taxa de crescimento entre os meses para identificar tendências de mercado.

## 🛠️ Tecnologias e Conceitos SQL
* **CTEs (Common Table Expressions):** Para organização e legibilidade do código.
* **Window Functions (LAG):** Para comparação de métricas entre diferentes períodos.
* **Agregações Complexas:** Para consolidação de valores de pedidos e pagamentos.

## 📈 Principais Insights
* Cálculo automatizado do faturamento total por mês.
* Identificação de meses com maior taxa de crescimento (MoM).
* Base preparada para alimentação de dashboards no Power BI.
